# DigitalContents - Galaxy Wars
> * #### 장르 : RTS 
> * #### 역할 : 프로그래머(인게임 구현)
> * #### 개발 언어/툴 : C# / Unity3d
> * #### 해당 파일 사용법 : 

## 프로젝트 소개
> ### 게임 소개
> * 해당 게임은 스타크래프트를 직관적이고 가볍게 플레이 가능하도록 제작한 웹게임으로 온 우주를 배경으로 하는 4인 멀티플레이 RTS 게임입니다.
> * 은하와 우주선의 업그레이드, 유닛생산, 특수 스킬
> ### 메인 시스템
> ### 게임 화면
